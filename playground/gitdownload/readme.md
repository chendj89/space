#### gitdownload

封装 git clone --depth 1

#### 安装

```bash
npm i @chencc/gitdownload
```

##### require 模式

```js
const git = require("@chencc/gitdownload");
git();
```

#### import 模式

```ts
import git from "@chencc/gitdownload";
git();
```

#### tsconfig

```json
{
  "types": ["@chencc/gitdownload"]
}
```
